Introduction

Welcome to the technical documentation for the C programming language. C is a powerful and versatile programming language that has been widely used for over four decades. This documentation will provide you with a comprehensive understanding of the language, its syntax, and key concepts.

printf("Hello World!");

What is C?

C is a general-purpose, procedural programming language developed by Dennis Ritchie in the early 1970s at Bell Labs. It is known for its efficiency, low-level control, and portability. C has been the foundation for many other programming languages, and it continues to be a fundamental language in software development.

Key Features

C offers several key features that make it a popular choice for system programming, embedded systems, and application development:

  • Low-level memory manipulation
  • High performance
  • Portability
  • Extensive standard library
  • Procedural programming paradigm

In this documentation, we will explore the essential concepts of C, starting with variables.

Variables

Variables in C are used to store and manipulate data. They are crucial for any program as they allow you to work with values of different types. C supports various data types, including:

  • int: Integer data type
  • float: Floating-point data type
  • char: Character data type
  • double: Double-precision floating-point data type
int age; float salary; char grade;

Functions

Functions in C allow you to encapsulate a specific set of actions into a reusable block of code. They play a crucial role in modular programming and code organization. Here's how you declare and define a simple function in C:

// Function declaration
int add(int a, int b);
// Function definition
int add(int a, int b) { return a + b; }

Functions can have parameters and return values, making them versatile for solving a wide range of problems. You can also use library functions provided by C or create your own custom functions.

Control Structures

Control structures in C allow you to control the flow of your program. Key control structures include:

  • if-else: Conditional branching
  • for: Looping construct for iteration
  • while: Another looping construct
  • switch: Multi-branching based on a value
int age = 25;
if (age >= 18) {
printf("You are an adult.");
} else {
printf("You are a minor."); }

Pointers

Pointers are a unique and powerful feature in C that allow you to work with memory addresses directly. They are crucial for tasks like dynamic memory allocation and low-level operations. Here's a simple pointer declaration:

int *ptr;

This declares a pointer to an integer. Pointers can be used for various purposes, such as accessing the address of variables, passing data by reference, and creating dynamic data structures like linked lists.
